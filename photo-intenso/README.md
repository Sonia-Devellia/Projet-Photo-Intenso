# Photo Intenso

**Photo Intenso** est mon premier projet d'étude en développement web. Ce projet est une simple application en **HTML** et **CSS** qui présente des photos dans une interface élégante. C'est une manière pour moi de mettre en pratique les compétences que j'ai apprises en HTML et CSS. 

Je suis ravie de partager ce projet et je serais très heureuse que d'autres développeurs et passionnés de web y participent, le commentent ou l'améliorent.

## Description

Le but de ce projet est de créer une galerie de photos basique avec une interface simple mais esthétique. Ce projet m'a permis d'expérimenter avec les bases de la structure d'une page web (HTML) et de la mise en page (CSS).

## Installation

### Prérequis

- Un éditeur de texte (par exemple, VS Code, Sublime Text, etc.)
- Un navigateur web moderne pour visualiser le projet (Google Chrome, Firefox, etc.)

### Étapes pour installer

1. Clonez ce dépôt sur votre machine locale :
    ```bash
    git clone https://github.com/ton-utilisateur/photo-intenso.git
    ```

2. Ouvrez le dossier du projet avec votre éditeur de texte préféré.

3. Lancez simplement le fichier `index.html` dans votre navigateur pour voir la galerie de photos.

## Utilisation

1. **Visualiser les photos :** Une fois le fichier `index.html` ouvert dans votre navigateur, vous pourrez voir les photos affichées avec une mise en page simple et propre.

2. **Personnalisation :** Vous pouvez modifier les images ou ajuster le design en modifiant les fichiers HTML et CSS.
